## 决策

创建 [specs/0003-public-sources-and-evidence-workflow.md](../specs/0003-public-sources-and-evidence-workflow.md)，作为第三开发切片的范围、合同、边界和验收真源。

本阶段只允许维护 SPEC、真源索引和决策记录，不允许创建公开资料与证据工作流的业务代码，不允许安装解析、采集或证据生成相关依赖。

## 范围主线

SPEC 0003 聚焦：

- 公开 URL 来源登记；
- 本地辅助文件登记；
- 普通网页、公开 PDF、Word、TXT 的文本解析；
- 证据卡片候选生成；
- 证据卡片编辑、确认和拒绝；
- 证据来源位置追溯；
- 受限、失败和不支持来源的结构化错误。

## 明确不进入

- 任意网站爬虫；
- 搜索引擎检索；
- 登录、验证码、付费墙或访问控制绕过；
- 数据集字段分析；
- Python 执行；
- 实验大纲；
- Word/PPT 生成；
- 真实 DeepSeek 调用作为验收前置条件。
